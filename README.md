# Hunch Desktop

Hunch — настольное приложение для мониторинга баз данных: выполнение SQL-запросов по расписанию, визуализация результатов на приборной панели, алерты и уведомления.

---

## Требования

| Компонент | Версия |
|-----------|--------|
| ОС | Windows 10 / 11 (64-бит) |
| Python | 3.10+ (только для запуска из исходников) |

---

## Установка (рекомендуется)

1. Скачайте последний установщик `Hunch_vX.Y.Z_installer.exe` со страницы [релизов](https://github.com/Melodic-Gambit/Hunch/releases)
2. Запустите установщик и следуйте инструкциям
3. Приложение установится в `%ProgramFiles%\Hunch`
4. Настройки хранятся в `%APPDATA%\Hunch`

---

## Portable-версия

Если установка невозможна (корпоративные ограничения, USB-флешка):

1. Скачайте `Hunch_vX.Y.Z.zip` со страницы релизов
2. Распакуйте в любую папку
3. Запустите `Hunch.exe`

---

## Запуск из исходников

```powershell
# Клонировать репозиторий
git clone https://github.com/Melodic-Gambit/Hunch.git
cd support_system/support_system

# Установить зависимости
pip install -r requirements.txt

# Запустить
python main.py
```

---

## Сборка EXE

```powershell
# Автоматическая сборка (читает версию из текущей git-ветки)
.\build.ps1
```

Результат: `dist\Hunch\Hunch.exe` и архив `dist\Hunch_vX.Y.Z.zip`.

Для сборки установщика (требуется [Inno Setup 6+](https://jrsoftware.org/isinfo.php)):

```powershell
iscc Hunch.iss
```

## Структура конфигурации

```
support_system/
├── config/          ← JSON-конфиги подключений к БД
├── queries/         ← SQL-файлы запросов
├── settings.json    ← Настройки приложения
└── logs/            ← Файлы логов
```

---

## Основные возможности

- **Приборная панель** — до 6 фреймов, 6 шаблонов компоновки, перетаскивание фреймов
- **Авто-обновление** — запросы выполняются по настраиваемому интервалу
- **Визуализация** — таблицы, графики, анимированные индикаторы, счётчики
- **Алерты** — звуковые и визуальные уведомления при изменении данных или срабатывании порога
- **GF. Scraping** — парсинг внешних источников с экспортом в Excel
- **Тема** — светлая/тёмная, настраиваемая цветовая схема
