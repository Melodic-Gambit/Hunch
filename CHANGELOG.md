# Changelog — Hunch Desktop

Все заметные изменения фиксируются в этом файле.  
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

---

## [v5.4.0] — 2026-05-22

### Добавлено
- **UX-05** Кнопка «Экспорт CSV» в диалоге «Статистика выполнения запросов» (`gui.py`)
- **TEST-01** Тесты для `stats_manager.py`: 15 тестов — `record`, `get_summary`, `get_recent`, `clear`, `rotate`, `rotate_by_size`, `get_db_size_kb`
- **TEST-03** Граничные тесты для `test_data_manager.py`: `_sanitize_name` со спецсимволами, `add_new_db` с пустым именем и переопределением дефолтов, `add_new_query` при отсутствующей директории

### Изменено
- **PERF-02** `db_manager.py`: TTL-пул соединений (30 сек) — соединение переиспользуется вместо открытия заново на каждый запрос
- **BUGFIX** `log_manager.py`: `add_log` теперь всегда сохраняет на диск — устранена потеря логов при падении приложения (было lazy flush каждые 50 записей)
- Переименован `support_system.ico` → `Hunch.ico`; все ссылки в коде обновлены

## [v5.3.0] — 2026-05-22

### Добавлено
- **REL-07** GitHub Actions workflows: `ci.yml` (pytest на каждый push/PR в `master` и `task/**`) и `release.yml` (сборка EXE + публикация GitHub Release по тегу `v*.*.*`)
- **REL-08** `.spec`-файлы (`main.spec`, `main_portable.spec`) добавлены в git-отслеживание: исключения `!main.spec` / `!main_portable.spec` в `.gitignore`
- **REL-09** `requirements.txt` дополнен: `matplotlib`, `pandas`, `requests`, `beautifulsoup4`, `pyperclip`
- **REL-10** `build.ps1`: автоматический вызов `iscc /DMyAppVersion=$version Hunch.iss` после ZIP-архивирования
- **REL-11** `Hunch.iss`: версия больше не захардкожена — читается из параметра `iscc /DMyAppVersion=X.Y.Z`, fallback `#ifndef MyAppVersion "0.0.0"`

## [v4.0.0] — 2026-05-21

### Добавлено
- **REL-01** Скрипт сборки `build.ps1`: автоматическое определение версии из git-ветки → обновление `version.txt` и `version_info.txt` → PyInstaller → ZIP-архив
- **REL-02** Установщик Inno Setup (`Hunch.iss`): установка в `%ProgramFiles%\Hunch`, ярлыки меню «Пуск» и рабочего стола, uninstaller
- **REL-03** Система версионирования: `version.txt`, `version_info.txt`, Windows EXE Properties (FileVersion, ProductVersion), заголовок окна `Hunch v{версия}`, версия в гамбургер-меню
- **REL-04** Многоразмерная иконка (16 × 16, 32 × 32, 48 × 48, 64 × 64, 128 × 128, 256 × 256) для корректного отображения в taskbar и Alt+Tab
- **REL-05** `CHANGELOG.md` и `README.md`
- **REL-06** Проверка обновлений при запуске через GitHub Releases API; настройка отключения в разделе «Обновления»
- **UX-10** Гибкая компоновка приборной панели: 6 шаблонов (Авто, Столбец, Строка, 1+2, 2+1, 2×2), диалог выбора шаблона, кнопка «Равные размеры», перетаскивание фреймов между ячейками
